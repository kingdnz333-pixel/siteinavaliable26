# Components

Este diretório é destinado a componentes reutilizáveis de UI (HTML/CSS/JS) que podem ser importados ou incluídos em páginas.

Atualmente o projeto é um site estático de uma única página, e as seções/camadas de layout estão construídas diretamente em `index.html`.

No futuro, se o projeto for convertido para um framework (React, Vue, etc.), aqui é onde os componentes individuais (Botão, Card, Formulário) podem ser organizados.
